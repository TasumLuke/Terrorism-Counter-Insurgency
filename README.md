# Rebel Factionalization and Conflict Duration

Replication for *How Does Rebel Factionalization Shape Conflict Duration? Evidence from a Natural Experiment in Northeast India, 1966-2022.*

The paper compares two structurally similar insurgencies. Mizoram got a single negotiating track and a durable 1986 accord. Nagaland got fragmentation-and-containment and 75 years of unresolved conflict. We build a Rebel Cohesion Index, run it against the MHA data, and find that a one-SD drop in cohesion predicts 33.7 percent more annual incidents. Cohesion mediates 68 percent of the effect.
